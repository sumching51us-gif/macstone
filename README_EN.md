# AnyStone

This project is a modernized refactor of [MegSpot](https://github.com/MegEngine/MegSpot).

For features and usage, please refer to the MegSpot official documentation.

## Install (macOS)

```bash
curl -fsSL https://raw.githubusercontent.com/sumching51us-gif/macstone/main/scripts/install.sh | bash
```

After installation, open the app from Launchpad or Spotlight (Cmd+Space).

If you see a security warning on first launch, go to **System Settings → Privacy & Security → Open Anyway**.
